# Face Mask Detection (ML Semester Project)

Проект по бинарной классификации изображений: определение наличия маски на лице. Датасет расположен в `data/with_mask` и `data/without_mask`.

## Структура
- `data/` — исходные изображения (маска/без маски)
- `src/` — исходный код (обучение, инференс, утилиты)
- `scripts/` — вспомогательные скрипты (смоук-тест и пр.)
- `notebooks/` — EDA, эксперименты
- `models/` — сохранённые веса
- `reports/` — отчёты, графики, презентация

## Быстрый старт
1. Установите зависимости (см. `requirements.txt`).
2. Запустите смоук-тест: он проверит доступность данных и чтение изображений.

## План
- Базовая модель: Transfer Learning (MobileNetV2/ResNet18)
- Метрики: Accuracy, Precision, Recall, F1, ROC-AUC, Confusion Matrix
- Аугментации, разделение на train/val/test, визуализации

## Лицензия и академическая честность
Использовать датасет только в рамках учебного проекта; соблюдать правила академической честности курса.
